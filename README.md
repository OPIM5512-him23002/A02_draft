# Project Overview
This project trains a neural network model (MLP Regressor) to predict median house values in California using the California Housing dataset from scikit-learn. The available features in the dataset are scaled and passed into the model with early stopping mechanism and some customized hyperparameters. Model performance is evaluated using predicted vs actual scatter plots for both train and test partitions.
